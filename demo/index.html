Textarea Demo

Cross browser textarea munipulator library written in CoffeeScript

See Richarea if you want to munipulate richarea (iframe) insted of textarea and see Jencil if you want to use JavaScript Editor which can handle Markup and WYSIWYG.

API Documents

<textarea id="textarea"></textarea>
<script type="text/javascript">
    textarea = new Textarea(document.getElementById('textarea'));
</script>

textarea.getValue();

Get value of textarea. Compatible as the code below

document.getElementById('textarea').value;

Demo

textarea.getSelection();

Get start and end index of text selection of textarea.

Demo

textarea.setSelection(start, end);

Set start and end index of text selection of textarea.

Demo

textarea.getSelected();

Get selected text of textarea.

Demo

textarea.replaceSelected(str, select=false);

Replace selected text with str. Reselect replaced text after replace if select is true

Demo

textarea.insertBeforeSelected(str, select=false);

Insert str before selected text. Reselect inserted text after insert if select is true

Demo

textarea.insertAfterSelected(str, select=false);

Insert str after selected text. Reselect inserted text after insert if select is true

Demo

textarea.wrapSelected(before, after, select=false, additional=undefined);

Wrap selected text with str. Reselect wrapped text after wrap if select is true

When additional is set, additional will wrapped when empty string is selected.
